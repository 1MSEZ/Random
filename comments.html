class/identifiers

all classess and indentifiers have served their own purpose.

#content used to display page content.
#footer used for display footer.
.info info sections for all contents.
.header used to change font size/weight for header.
paddings and text alignments are also changed.
.top used to change font size and included text shadow for header.
.nav sub classes includes: li, a and a:hover.
used to gain effects on selecting nav bar items, change colour when hovering over, selected.
